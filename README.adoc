= Blue Archive ASMR

Translations/Subtitles for Blue Archive ASMR voice works.

This repository contains spoilers.
Please proceed at your own risk.

== Usage

These files are intended to be used with https://mpv.io[mpv media player], but other players should work.
Place the appropriate subtitle files in the same directory as your audio files, and `mpv` should detect them when you listen to the ASMR voice pack.
